# DeepSeek V4 Pro 能力测试提示词

## 提示词（直接复制使用）

---

请用 **单个 HTML 文件** 实现一个完整的「星际塔防」H5 游戏，要求如下：

### 核心玩法
1. 地图为 15×10 的网格，敌人沿预设路径（S形蛇形路线）从左上角走到右下角
2. 玩家可在非路径格子上放置 3 种防御塔：
   - 🔫 机枪塔（攻速快、伤害低、射程近，价格100）
   - 🚀 火箭塔（攻速慢、范围伤害、射程远，价格200）
   - ⚡ 电磁塔（减速敌人50%持续2秒、中等伤害，价格150）
3. 敌人有 3 种类型：
   - 普通兵（血量100、速度中等）
   - 坦克（血量400、速度慢、对火箭塔伤害减免30%）
   - 飞行兵（血量80、速度快、只有火箭塔能攻击）
4. 每波敌人数量和强度递增，共 20 波，每 5 波出一个 BOSS（血量为普通的10倍）

### 技术要求
1. 纯原生 JS + Canvas 渲染，禁止任何外部依赖
2. 实现 A* 或类似寻路算法让敌人沿路径移动
3. 防御塔自动锁定最近/血量最低的敌人（可切换策略）
4. 子弹需要有飞行轨迹动画（不是瞬间命中）
5. 实现粒子效果：爆炸、电磁脉冲、敌人死亡
6. 60fps 流畅运行，使用 requestAnimationFrame
7. 使用对象池模式管理子弹和粒子，避免 GC 卡顿

### UI 要求
1. 顶部状态栏：波次、金币、生命值、FPS 计数器
2. 右侧面板：防御塔选择、升级（3级，每级属性提升40%）、出售（返还60%金币）
3. 底部：开始波次按钮、游戏速度切换（1x/2x/3x）
4. 选中防御塔时显示攻击范围圆圈
5. 敌人头顶显示血条
6. 整体采用深色科幻风格 UI，带霓虹发光效果

### 加分项
1. 本地存储最高分记录
2. 音效（用 Web Audio API 合成，不用外部音频文件）
3. 开场动画和游戏结束画面
4. 手机端触摸适配

请直接输出完整可运行的 HTML 文件，不要分段，不要省略任何代码。

## 评分维度

| 维度 | 权重 | 评判标准 |
|------|------|----------|
| 可运行性 | 30% | 直接保存为 .html 能否打开即玩，无报错 |
| 功能完整度 | 25% | 3种塔、3种敌人、20波、升级系统是否都实现 |
| 代码质量 | 15% | 架构是否清晰、有无对象池、性能优化 |
| 视觉效果 | 15% | 粒子效果、UI美观度、动画流畅度 |
| 加分项 | 15% | 音效、存储、手机适配等额外功能 |

## 使用建议

1. 把提示词直接发给 DeepSeek V4 Pro
2. 将输出保存为 `game.html`，浏览器打开测试
3. 同时发给 Claude/GPT/Gemini 做横向对比
4. 重点观察：是否一次生成就能运行、代码量和完成度、是否偷懒省略代码
